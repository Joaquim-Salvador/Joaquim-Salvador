Sou estudante de **Engenharia de Software pela Univille** (Joinville/SC) e **Técnico em Desenvolvimento de Software pelo SENAI**. Tenho foco em desenvolvimento back-end com Java e Spring Boot, construindo APIs REST robustas e integrações com serviços externos.

Estou em busca da minha **primeira oportunidade profissional** como desenvolvedor (estágio ou júnior), trazendo disciplina técnica, projetos reais no portfólio e vontade genuína de crescer dentro de um time.

## 📂 Projetos

### 🔗 [Voltaki](https://github.com/Joaquim-Salvador/voltaki)
> API REST para uma plataforma de reconciliação romântica com mensagens personalizadas, slideshows de fotos e músicas de fundo.

**Stack:** Java · Spring Boot · PostgreSQL · Gemini AI (Google)  
**Destaques:** Integração com IA generativa, geração de conteúdo personalizado via API, arquitetura RESTful com DTOs e camadas bem separadas.

---

### 🔗 [Email Classifier](https://github.com/Joaquim-Salvador/email-classifier)
> Classificador de e-mails que identifica padrões em texto utilizando técnicas de processamento de linguagem natural.

**Stack:** Python  
**Destaques:** Análise e classificação automática de texto, identificação de padrões em datasets de e-mail.

---

### 🔗 [Certificate Generator](https://github.com/Joaquim-Salvador/certificate-generator)
> Sistema web para geração automatizada de certificados com preenchimento dinâmico de dados.

**Stack:** Angular · TypeScript  
**Destaques:** Interface responsiva, geração de documentos dinâmicos, integração de dados via formulário.
